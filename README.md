# الوعي الجماعي كنظام ديناميكي غير خطي

---

## الوصف

المشروع عبارة عن أرشيف بحثي يحاول إعادة تعريف الأسواق المالية.  
يتجاوز هذا المشروع النماذج التقليدية القائمة على البيانات الاقتصادية، العوامل الجيوسياسية، وغيرها من المؤشرات الخارجية،  
ويتجه نحو فهم الأسواق كنظام ديناميكي غير خطي.

يقوم المشروع بترجمة الوعي الجمعي للمنشآت والأفراد،  
حيث يتحول عدم الاتفاق على السعر إلى حركات سوقية قابلة للرصد،  
مبرزًا الهيكلية التي تكشف السبب والنتيجة.

المشروع مفتوح المصدر للأغراض البحثية والتعليمية فقط
